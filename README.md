# 🏢 Sistema de Registro de Visitantes - SLB

Aplicación web para el registro digital de visitantes, diseñada para reemplazar formatos físicos y mejorar la trazabilidad, seguridad y gestión de datos en campo.

## 🚀 Características

- 📋 Formulario web responsivo
- ✍️ Captura de firma digital (canvas)
- 💻 Registro de ingreso de laptops
- 🔐 Validación de aceptación de tratamiento de datos
- ☁️ Almacenamiento en Supabase
- 📊 Dashboard administrativo
- 📥 Exportación a Excel con formato profesional

## 🛠️ Tecnologías utilizadas

- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Base de datos: Supabase (PostgreSQL)
- Librerías:
  - openpyxl (Excel)
  - Pillow (imágenes)
  - requests

## 📊 Funcionalidades del Dashboard
- Total de registros
- Descarga de Excel con:
- Encabezados personalizados
    - ID automático
    - Firma como imagen
    - Fechas en horario Colombia

## 📌 Notas
- La firma se guarda en formato Base64
- El Excel incluye formato corporativo (estilo SLB)
- Compatible con dispositivos móviles
